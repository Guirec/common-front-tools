# Alert

The alert component is made to highlight content with a context according to the chosen color.

There are several color choices:

| Name        | Dominant color | Modifier class       |
|:-----------:|:--------------:|:--------------------:|
| Default     | Black          |                      |
| Information | Blue           | `c-alert--info`      |
| Warning     | Orange         | `c-alert--warning`   |
| Error       | Red            | `c-alert--error`     |
| Success     | Green          | `c-alert--success`   |
