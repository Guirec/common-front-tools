# Button

Button class can be applied on `<button>`, `<a>` or `<input>` html tags. Several variants and state are available.

If there is an attribute `disabled`, the style adjust accordingly, for all the variants and states.

## Variants

| Variant  |        Class         |
| :------: | :------------------: |
| Reversed | `c-button--reversed` |
| Primary  | `c-button--primary`  |

## State

|  State  |    Class     |
| :-----: | :----------: |
| Loading | `is-loading` |
