# Container

The `o-container` class can be used to create a frame where the content into it will be centered if the screen size is above the maximum width set, with some padding on each side. The maximum width is actually set to **144rem (~1440px)** with a right and left padding of **1.6rem (~16px)**.

```html
<div class="o-container">
  <p>Some content</p>
</div>
```
