# Typography

We want to use the system font as font family, to do that we can use `system-ui` as value.

The goal is to list the font properties, like sizes and weight to avoid using a property that is outside our scope.

## Font weights

| Label    | Value |
| :-------:|:-----:|
| light    | 300   |
| regular  | 400   |
| semibold | 600   |
| bold     | 700   |

## Spacing

Line height must be define unitless and the base value applied is `1.5`.
