# Display

Some elements or containers doesn't require to have an object, pattern or composant created just for them. In that case, display classes will help to order content, if enabled.

|         Corresponding CSS         |          Class          |
| :-------------------------------: | :---------------------: |
|         `display: block;`         |        `u-block`        |
|         `display: none;`          |        `u-none`         |
|        `display: inline;`         |       `u-inline`        |
|     `display: inline-block;`      |    `u-inline-block`     |
|         `display: flex;`          |        `u-flex`         |
|        `flex-wrap: wrap;`         |      `u-flex-wrap`      |
|    `flex-wrap: wrap-reverse;`     |  `u-flex-wrap-reverse`  |
|       `flex-wrap: nowrap;`        |     `u-flex-nowrap`     |
|      `flex-direction: row;`       |      `u-flex-row`       |
|  `flex-direction: row-reverse;`   |  `u-flex-row-reverse`   |
|     `flex-direction: column;`     |     `u-flex-column`     |
|  `justify-content: flex-start;`   |   `u-flex-main-start`   |
|   `justify-content: flex-end;`    |    `u-flex-main-end`    |
|    `justify-content: center;`     |  `u-flex-main-center`   |
| `justify-content: space-between;` |  `u-flex-main-between`  |
| `justify-content: space-around;`  |  `u-flex-main-around`   |
|    `align-items: flex-start;`     |  `u-flex-cross-start`   |
|     `align-items: flex-end;`      |   `u-flex-cross-end`    |
|      `align-items: center;`       |  `u-flex-cross-center`  |
|     `align-items: baseline;`      | `u-flex-cross-baseline` |
|      `align-items: stretch;`      | `u-flex-cross-stretch`  |
|            `flex: 1;`             |   `u-flex-item-fluid`   |
|           `flex: auto;`           |   `u-flex-item-auto`    |
|         `flex: initial;`          |  `u-flex-item-initial`  |
|           `flex: none;`           |   `u-flex-item-none`    |
|           `order: -1;`            |   `u-flex-item-first`   |
|            `order: 0;`            |  `u-flex-item-medium`   |
|            `order: 1;`            |   `u-flex-item-last`    |
