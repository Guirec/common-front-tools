# Widths

If enabled, utility classes are generated according to the widths values in the settings.

|  Value   |   Class    |
| :------: | :--------: |
|   25%    | `u-w-1/4`  |
| 33.3333% | `u-w-1/3`  |
|   50%    | `u-w-1/2`  |
| 66.6667% | `u-w-2/3`  |
|   75%    | `u-w-3/4`  |
|   100%   | `u-w-full` |
|   auto   | `u-w-auto` |
