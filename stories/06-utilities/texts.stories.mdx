# Texts

If enabled, utility classes are generated according to the texts values in the settings.

## Font sizes

|     Value     |    Class     |
| :-----------: | :----------: |
| 1.2rem (12px) | `u-text-xxs` |
| 1.4rem (14px) | `u-text-xs`  |
| 1.6rem (16px) | `u-text-sm`  |
| 1.8rem (18px) | `u-text-md`  |
|  2rem (20px)  | `u-text-lg`  |
| 2.2rem (22px) | `u-text-xl`  |
| 2.4rem (24px) | `u-text-2xl` |
| 2.6rem (26px) | `u-text-3xl` |
| 2.8rem (28px) | `u-text-4xl` |
|  3rem (30px)  | `u-text-5xl` |
| 3.2rem (32px) | `u-text-6xl` |

## Font weight

|  Value   |       Class       |
| :------: | :---------------: |
|  Light   |  `u-text-light`   |
| Regular  | `u-text-regular`  |
| Semibold | `u-text-semibold` |
|   Bold   |   `u-text-bold`   |

## Text alignement

|  Value  |      Class       |
| :-----: | :--------------: |
|  Left   |  `u-text-left`   |
|  Right  |  `u-text-right`  |
| Center  | `u-text-center`  |
| Justify | `u-text-justify` |
